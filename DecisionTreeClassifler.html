3. Contoh Threshold Split:#

Threshold = 23 cm

  • Group Left (‚â§ 23)

  • Group Right (> 23)

4. Entropy Awal (sebelum split)#

Tiap kelas jumlahnya sama:

\[ p_1 = p_2 = p_3 = \frac{5}{15} = \frac{1}{3} \]

Maka:

\[ Entropy(D) = -\sum_{i=1}^3 \left(\frac{1}{3} \log_2 \frac{1}{3} \right) = -3 \cdot \frac{1}{3} \log_2 \frac{1}{3} = \log_2 3 \approx \boxed{1.585} \]

5. Entropy Setelah Split#

üìå Group Left (‚â§ 23):#

Data: No 6‚Äì10, 11‚Äì15 ‚Üí Total 10 data

  • Cucumis anguria = 5

  • Cucumis sativus = 5

  • Cucumis melo = 0

Probabilitas:

  • \(p_1 = 5/10 = 0.5\)

  • \(p_2 = 5/10 = 0.5\)

  • \(p_3 = 0\)

Entropy:

\[ Entropy(left) = -0.5 \log_2 0.5 - 0.5 \log_2 0.5 = 1.0 \]

üìå Group Right (> 23):#

Data: No 1‚Äì5 ‚Üí Total 5 data

  • Cucumis melo = 5

  • Lainnya = 0

Entropy:

\[ Entropy(right) = -1 \log_2 1 = 0 \]

6. Information Gain:#

\[ IG = Entropy(D) - \left( \frac{10}{15} \cdot Entropy(left) + \frac{5}{15} \cdot Entropy(right) \right) \]
\[ IG = 1.585 - \left( \frac{10}{15} \cdot 1 + \frac{5}{15} \cdot 0 \right) = 1.585 - \frac{10}{15} = \boxed{0.918} \]

7. Gini Index#

üìå Gini Awal:#

\[ Gini(D) = 1 - \sum p_i^2 = 1 - 3 \cdot \left(\frac{1}{3}\right)^2 = 1 - \frac{1}{3} = \boxed{0.667} \]

üìå Gini Left (‚â§ 23):#

  • 5 anguria, 5 sativus

  • \(p = 0.5\) untuk keduanya

\[ Gini(left) = 1 - 0.5^2 - 0.5^2 = 1 - 0.25 - 0.25 = \boxed{0.5} \]

üìå Gini Right (> 23):#

  • 5 melo

\[ Gini(right) = 1 - 1^2 = 0 \]

8. Gini Split:#

\[ Gini_{split} = \frac{10}{15} \cdot 0.5 + \frac{5}{15} \cdot 0 = \boxed{0.333} \]

2. Ambil Threshold Split#

Pilih threshold = 9 cm (Terjadi perpindahan dari anguria ‚Üí melo)

3. Komposisi Data#

Sebelum Split:

  • 5 Cucumis melo

  • 5 Cucumis sativus

  • 5 Cucumis anguria

Total data = 15

\[ Entropy(D) = -3 \cdot \left(\frac{5}{15} \cdot \log_2 \frac{5}{15}\right) = -3 \cdot \left(\frac{1}{3} \cdot \log_2 \frac{1}{3}\right) = \log_2 3 \approx \boxed{1.585} \]

4. Split Berdasarkan Threshold (‚â§ 9 cm dan > 9 cm)#

Group Left (‚â§ 9 cm):#

Data No: 6‚Äì15 (10 data)

  • 5 Cucumis sativus

  • 5 Cucumis anguria

\[ Entropy(left) = -\left(\frac{5}{10} \cdot \log_2 \frac{5}{10}\right) - \left(\frac{5}{10} \cdot \log_2 \frac{5}{10}\right) = -2 \cdot (0.5 \cdot -1) = \boxed{1.0} \]

Group Right (> 9 cm):#

Data No: 1‚Äì5 (5 data)

  • 5 Cucumis melo

\[ Entropy(right) = -1 \cdot \log_2 1 = \boxed{0} \]

5. Information Gain#

\[\begin{split} IG = Entropy(D) - \left( \frac{10}{15} \cdot 1.0 + \frac{5}{15} \cdot 0 \right) \\ = 1.585 - \left( \frac{10}{15} \cdot 1.0 \right) = \boxed{0.918} \end{split}\]

6. Gini Index#

Gini(D) ‚Äî Sebelum Split:#

\[ Gini(D) = 1 - 3 \cdot \left(\frac{1}{3}\right)^2 = 1 - \frac{3}{9} = \boxed{0.667} \]

Gini(Left):#

  • 5 Cucumis sativus

  • 5 Cucumis anguria

\[ Gini(left) = 1 - 0.5^2 - 0.5^2 = 1 - 0.25 - 0.25 = \boxed{0.5} \]

Gini(Right):#

  • 5 Cucumis melo

\[ Gini(right) = 1 - 1^2 = \boxed{0} \]

Gini Split:#

\[ Gini_{split} = \frac{10}{15} \cdot 0.5 + \frac{5}{15} \cdot 0 = \boxed{0.333} \]

Visualisasi Decision Tree#

Ini adalah code program untuk mencari decision tree sekaligus rules

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
from sklearn.tree import DecisionTreeClassifier, plot_tree, export_text
from sklearn.preprocessing import LabelEncoder

# === Data Buah Cucumis ===
data = {
    "Panjang Buah (cm)": [25, 28, 30, 27, 29, 20, 22, 21, 19, 23, 15, 14, 16, 15, 17],
    "Diameter Buah (cm)": [15, 16, 17, 14, 16, 5, 6, 5.5, 4.5, 6, 8, 7.5, 8.5, 8, 9],
    "Kelas": [
        "Cucumis melo", "Cucumis melo", "Cucumis melo", "Cucumis melo", "Cucumis melo",
        "Cucumis sativus", "Cucumis sativus", "Cucumis sativus", "Cucumis sativus", "Cucumis sativus",
        "Cucumis anguria", "Cucumis anguria", "Cucumis anguria", "Cucumis anguria", "Cucumis anguria"
    ]
}
df = pd.DataFrame(data)

# === Label Encoding ===
le = LabelEncoder()
df['Label'] = le.fit_transform(df['Kelas'])  # anguria=0, melo=1, sativus=2 (urutan tergantung sklearn)

# === Fitur dan Target ===
X = df[["Panjang Buah (cm)", "Diameter Buah (cm)"]]
y = df["Label"]  # gunakan label numerik hasil encoding

# === Train Decision Tree ===
clf = DecisionTreeClassifier(criterion="gini", max_depth=3, random_state=42)
clf.fit(X, y)

# === 1. Tampilkan Pohon Keputusan ===
print("\n=== Pohon Keputusan (Decision Tree) ===")
tree_rules = export_text(clf, feature_names=list(X.columns))
print(tree_rules)

=== Pohon Keputusan (Decision Tree) ===
|--- Panjang Buah (cm) <= 18.00
|   |--- class: 0
|--- Panjang Buah (cm) >  18.00
|   |--- Diameter Buah (cm) <= 10.00
|   |   |--- class: 2
|   |--- Diameter Buah (cm) >  10.00
|   |   |--- class: 1

Visualisasi Splitting#

# === Uji 2 Data Baru ===
uji_baru = pd.DataFrame({
    'Panjang Buah (cm)': [18, 26],
    'Diameter Buah (cm)': [6, 15]
})

prediksi = clf.predict(uji_baru)
hasil = le.inverse_transform(prediksi)

print("\n=== Prediksi 2 Data Baru ===")
for i, (p, d) in enumerate(zip(uji_baru['Panjang Buah (cm)'], uji_baru['Diameter Buah (cm)'])):
    print(f"Data {i+1}: Panjang={p}, Diameter={d} ‚Üí Prediksi: {hasil[i]}")

# === Visualisasi Decision Boundary dengan Garis Split ===
print("\n=== Visualisasi Decision Boundary ===")
plot_colors = ["r", "y", "g"]
plot_step = 0.2

x_min, x_max = X['Panjang Buah (cm)'].min() - 2, X['Panjang Buah (cm)'].max() + 2
y_min, y_max = X['Diameter Buah (cm)'].min() - 1, X['Diameter Buah (cm)'].max() + 1

xx, yy = np.meshgrid(np.arange(x_min, x_max, plot_step),
                     np.arange(y_min, y_max, plot_step))

grid = np.c_[xx.ravel(), yy.ravel()]
grid_df = pd.DataFrame(grid, columns=['Panjang Buah (cm)', 'Diameter Buah (cm)'])
Z = clf.predict(grid_df)
Z = Z.reshape(xx.shape)

plt.figure(figsize=(8, 6))

# Area warna background untuk masing-masing kelas
plt.contourf(xx, yy, Z, cmap=plt.cm.RdYlBu, alpha=0.6)

# Garis pemisah antar kelas (split boundary)
plt.contour(xx, yy, Z, levels=np.unique(Z), linewidths=1.5, colors='k')

# Plot titik data training
for label, color in zip(np.unique(y), plot_colors):
    idx = np.where(y == label)
    plt.scatter(X.iloc[idx]['Panjang Buah (cm)'], X.iloc[idx]['Diameter Buah (cm)'],
                c=color, edgecolor='k', label=le.inverse_transform([label])[0])

plt.xlabel("Panjang Buah (cm)")
plt.ylabel("Diameter Buah (cm)")
plt.title("Decision Boundary (Panjang vs Diameter Buah)")
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

=== Prediksi 2 Data Baru ===
Data 1: Panjang=18, Diameter=6 ‚Üí Prediksi: Cucumis anguria
Data 2: Panjang=26, Diameter=15 ‚Üí Prediksi: Cucumis melo
